## 1. Objectifs <a name='objectifs'>

L’objectif de ce projet est d’analyser le lien entre les infrastructures présentes dans les communes françaises et leur orientation politique. Plus précisément, il s’agit d’étudier dans quelle mesure l’accès aux infrastructures sportives, de loisirs ou culturelles est associé aux scores de La France Insoumise et du Rassemblement National lors de l’élection présidentielle de 2022.

## 2. Données <a name='donnees'>

Nous nous sommes appuyées essentiellement sur des données publiques : 
- Les résultats au premier tour des élections présidentielles de 2022 étaient fournis par data.gouv. 
- La base de données concernant les équipements des communes françaises était fournie par l'Insee. 
- Les données spatiales utilisées pour la représentation de certains résultats sont issus de l'IGN. 
- Nous nous sommes également servies de la document du Code Officiel Géographique (COG) de l'Insee par moment.
